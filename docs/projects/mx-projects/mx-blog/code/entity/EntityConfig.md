
  EntityConfig - 实体配置
===============================================================================

> 实体配置：补充实体的作用。
拥有一些额外的配置，并对应有一些可选项，无配置时，可进行默认；
不涉及时间变量；

## 名词，动词，或组合词

### 名词

####  使用场景

- 是什么

### 动词

#### 使用方式

####  使用场景

- 是否做什么

### 组合词


### 使用方式

- not + v.++
- non + n.


#### 使用场景

- 不允许做什么
